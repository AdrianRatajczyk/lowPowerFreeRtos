00291 /* 
00292  * xSemaphoreAltTake() is an alternative version of xSemaphoreTake().
00293  *
00294  * The source code that implements the alternative (Alt) API is much 
00295  * simpler  because it executes everything from within a critical section.  
00296  * This is  the approach taken by many other RTOSes, but FreeRTOS.org has the 
00297  * preferred fully featured API too.  The fully featured API has more 
00298  * complex  code that takes longer to execute, but makes much less use of 
00299  * critical sections.  Therefore the alternative API sacrifices interrupt 
00300  * responsiveness to gain execution speed, whereas the fully featured API
00301  * sacrifices execution speed to ensure better interrupt responsiveness.
00302  */
00303 #define xSemaphoreAltTake( xSemaphore, xBlockTime )     xQueueAltGenericReceive( ( xQueueHandle ) ( xSemaphore ), NULL, ( xBlockTime ), pdFALSE )
